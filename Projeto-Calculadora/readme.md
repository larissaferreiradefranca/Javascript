# Calculadora JavaScript

Projeto de estudo focado em manipulação do DOM, eventos e lógica de programação com JavaScript vanilla.

## Objetivos de aprendizado

- Capturar e processar input do usuário
- Implementar event listeners
- Manipular o DOM de forma eficiente
- Trabalhar com operações matemáticas e validação de dados
- Tratamento de erros (divisão por zero)

## Tecnologias

- HTML5 semântico
- CSS3 para estilização
- JavaScript ES6+ (vanilla)

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Validação de divisão por zero
- Interface responsiva
- Feedback visual de resultado

## Conceitos aplicados

**JavaScript:**
- `querySelector()` para seleção de elementos
- `addEventListener()` para captura de eventos
- Destructuring assignment `[a, b]`
- Arrow functions
- Template literals
- Validação condicional

## Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Teste as operações matemáticas

## Aprendizados

Este projeto me permitiu praticar:
- Como estruturar código JavaScript de forma organizada
- A importância de validar inputs do usuário
- Reutilização de código através de funções
- Manipulação eficiente do DOM
